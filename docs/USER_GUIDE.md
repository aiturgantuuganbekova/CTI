## 1. Что это за платформа

**Aiturgan CTI** (Cryptocurrency Trading Indicator) — это веб-платформа для анализа криптовалютного рынка в реальном времени.

### Что она делает:
- Получает данные о ценах криптовалют с биржи Binance
- Анализирует рынок с помощью технических индикаторов (RSI, MACD, EMA, Bollinger Bands)
- Генерирует торговые сигналы: **ПОКУПКА (BUY)**, **ПРОДАЖА (SELL)** или **ОЖИДАНИЕ (HOLD)**
- Позволяет протестировать стратегии на исторических данных (бэктестинг)
- Отправляет уведомления о сигналах в Telegram
- Сравнивает эффективность разных стратегий

### Что она НЕ делает:
- Не совершает реальные сделки на бирже
- Не гарантирует прибыль
- Не является финансовой рекомендацией

Платформа создана для **аналитики и обучения** — она помогает понять, как работают торговые стратегии, и оценить их эффективность.

## 3. Панель управления (Dashboard)

Dashboard — это главная страница, которая показывает общую картину рынка.

### Что на ней находится:

#### Верхняя панель
- **BTC/USDT** — текущая цена биткоина (обновляется каждые 10 секунд)
- **Процент изменения** — изменение цены за 24 часа (зелёный = рост, красный = падение)
- **Объём** — объём торгов за 24 часа

#### Карточки метрик (4 штуки)
| Карточка | Что показывает |
|---|---|
| **ROI (Доходность)** | Общая доходность ваших стратегий в процентах |
| **Винрейт** | Процент успешных сделок из общего количества |
| **Активные сигналы** | Количество сигналов, которые сейчас активны |
| **Всего сделок** | Общее количество сделок за всё время |

#### Выбор монеты
Кнопки для переключения между криптовалютами:
- **BTCUSDT** — Bitcoin
- **ETHUSDT** — Ethereum
- **BNBUSDT** — BNB
- **SOLUSDT** — Solana
- **XRPUSDT** — Ripple

#### Выбор таймфрейма
Определяет период одной свечи на графике:
| Таймфрейм | Описание |
|---|---|
| 5m | Каждая свеча = 5 минут |
| 15m | Каждая свеча = 15 минут |
| 30m | Каждая свеча = 30 минут |
| 1h | Каждая свеча = 1 час |
| 4h | Каждая свеча = 4 часа |
| 1d | Каждая свеча = 1 день |

> Короткие таймфреймы (5m, 15m) — для быстрой торговли (скальпинг).
> Длинные таймфреймы (4h, 1d) — для долгосрочного анализа.

#### Свечной график
Показывает движение цены выбранной монеты:
- **Зелёная свеча** — цена выросла за период
- **Красная свеча** — цена упала за период
- Тело свечи показывает разницу между ценой открытия и закрытия
- Тени (фитили) показывают максимум и минимум цены

#### Таблица последних сигналов
Показывает 5 последних сгенерированных сигналов с информацией:
- Время генерации
- Монета
- Использованная стратегия
- Тип сигнала (BUY/SELL/HOLD)
- Цена на момент сигнала
- Уровень уверенности (0-100%)

## 4. Стратегии (Strategy)

Страница Strategy — это основной инструмент для генерации торговых сигналов.

### Левая панель — Настройка

#### Выбор монеты
Какую криптовалюту анализировать (BTC/USDT, ETH/USDT и т.д.)

#### Выбор стратегии
Какой алгоритм использовать для анализа:

| Стратегия | Как работает | Когда использовать |
|---|---|---|
| **RSI** | Индекс относительной силы. Измеряет скорость и изменение ценовых движений (0-100). RSI < 30 = перепроданность (покупка), RSI > 70 = перекупленность (продажа) | Для определения разворотных точек |
| **MACD** | Схождение/расхождение скользящих средних. Когда линия MACD пересекает сигнальную линию вверх — покупка, вниз — продажа | Для определения направления тренда |
| **EMA** | Экспоненциальная скользящая средняя. Сравнивает короткую EMA(9) с длинной EMA(21). Короткая выше длинной — покупка | Для следования за трендом |
| **Bollinger Bands** | Полосы Боллинджера. Цена у нижней полосы — покупка, у верхней — продажа | Для торговли в диапазоне |
| **Combined** | Комбинированная. Использует все 4 стратегии и принимает решение голосованием большинства | Для наиболее надёжного сигнала |

#### Таймфрейм
На каком интервале анализировать (5m, 15m, 30m, 1h, 4h, 1d)

#### Уровень риска
- **Низкий риск** — консервативная торговля
- **Средний риск** — сбалансированный подход
- **Высокий риск** — агрессивная торговля

#### Стоп-лосс и тейк-профит
- **Стоп-лосс (%)** — при каком проценте убытка автоматически закрыть сделку (например, 2% = если цена упадёт на 2%, сделка закроется)
- **Тейк-профит (%)** — при каком проценте прибыли автоматически закрыть сделку (например, 3% = если цена вырастет на 3%, сделка закроется)

#### Кнопка "Генерировать сигнал"
Нажмите для получения анализа. Система:
1. Загрузит последние свечи с Binance
2. Рассчитает индикаторы
3. Определит сигнал (BUY / SELL / HOLD)
4. Покажет результат

### Правая панель — Результат анализа

После генерации сигнала показывается:
- **Тип сигнала** — BUY SIGNAL (зелёный), SELL SIGNAL (красный), HOLD SIGNAL (жёлтый)
- **Показатель точности** — круговая диаграмма с процентом уверенности (0-100%)
- **Описание** — текстовое пояснение почему такой сигнал

### Индикаторы

Под результатом показываются 4 панели с текущими значениями индикаторов:

| Панель | Что показывает |
|---|---|
| **RSI** | Текущее значение RSI (0-100). Зелёные бары = бычий рынок |
| **MACD** | Гистограмма MACD. Зелёные бары выше нуля = бычий тренд |
| **EMA Ribbon** | Значения EMA на разных периодах (20, 50, 200) |
| **Bollinger Bands** | Визуализация текущего положения цены относительно полос |

### Сохранённые пресеты

Внизу страницы можно сохранять конфигурации стратегий:
- Нажмите **"+ Новый пресет"** для создания
- Укажите монету, стратегию, таймфрейм, стоп-лосс, тейк-профит
- Включите переключатель **АВТО-ТОРГОВЛЯ** для автоматического мониторинга (сигналы будут генерироваться каждые 60 секунд)

## 5. Бэктестинг (Backtest)

Бэктестинг позволяет проверить стратегию на исторических данных — "Что было бы, если бы я торговал по этой стратегии в прошлом?"

### Как использовать:

1. **Выберите параметры:**
   - Монета (BTCUSDT, ETHUSDT и т.д.)
   - Стратегия (RSI, MACD, EMA, Bollinger Bands, Combined)
   - Таймфрейм (1h, 4h, 1d и т.д.)
   - Период тестирования (дата начала и конца)
   - Начальный баланс (например, $10,000)

2. **Нажмите "Запустить тест"**

3. **Дождитесь результатов** (обычно несколько секунд)

### Результаты бэктеста:

#### Метрики (5 карточек)
| Метрика | Что означает | Хорошее значение |
|---|---|---|
| **ROI %** | Доходность в процентах | > 0% (положительный) |
| **Винрейт** | Процент прибыльных сделок | > 50% |
| **Общий P/L** | Итоговая прибыль/убыток в долларах | Положительное число |
| **Макс. просадка** | Максимальное падение баланса от пика | < 20% |
| **Всего сделок** | Количество совершённых сделок | Зависит от стратегии |

#### Кривая доходности (Equity Curve)
График показывает как менялся ваш баланс во времени:
- **Растущая линия** — стратегия прибыльна
- **Падающая линия** — стратегия убыточна
- Вкладка **ПРОСАДКА** показывает падения от максимумов

#### Статистика эффективности
- **Профит-фактор** — отношение прибыли к убыткам. > 1.5 = хорошо
- **Средняя прибыль** — средний размер выигрышной сделки
- **Средний убыток** — средний размер проигрышной сделки
- **Коэффициент Шарпа** — доходность с учётом риска. > 1.0 = хорошо

#### История сделок
Таблица всех виртуальных сделок:
- Тип (BUY/SELL)
- Цена входа и выхода
- Прибыль/убыток
- Даты открытия и закрытия

> Кнопка **ЭКСПОРТ CSV** сохраняет историю сделок в файл для анализа в Excel.

### Интервал мониторинга (Monitoring Interval)
Позволяет настроить частоту автоматической проверки стратегий:

| Элемент | Описание |
|---|---|
| **Статус** | Зелёная точка + "Active — every Xs" или "Stopped" |
| **Поле ввода** | Интервал в секундах (минимум 5 секунд) |

| **Start / Stop** | Запустить или остановить мониторинг |

> Пример: установите 10 секунд — система будет проверять рынок и отправлять сигналы в Telegram каждые 10 секунд.

### Мониторинг стратегий
Список ваших сохранённых стратегий с переключателями вкл/выкл.
- **Включено** (зелёный) — стратегия участвует в автоматическом мониторинге
- **Выключено** (серый) — стратегия неактивна

> **Важно:** для работы мониторинга нужны **и** запущенный мониторинг (Monitoring Interval → Active), **и** хотя бы одна включённая стратегия.

## 8. Telegram-уведомления

Telegram-бот отправляет уведомления о торговых сигналах прямо в мессенджер.

### Настройка (один раз):

**Шаг 1.** Найдите бота **@aiturgan_cti_bot** в Telegram

**Шаг 2.** Отправьте команду `/start` боту

**Шаг 3.** Узнайте свой Chat ID:
- Отправьте любое сообщение боту **@userinfobot**
- Он ответит числом — это ваш Chat ID (например: `1034380014`)

**Шаг 4.** На сайте перейдите в **Настройки → Telegram**:
- Вставьте ваш Chat ID в поле
- Нажмите **Сохранить**
- Нажмите **Тест** для проверки — вам придёт тестовое сообщение

### Какие уведомления приходят:
- **BUY сигнал** — система рекомендует покупку (зелёный)
- **SELL сигнал** — система рекомендует продажу (красный)
- **HOLD сигнал** — рынок в неопределённости, ожидание (жёлтый)
- **Закрытие сделки** — когда сработал стоп-лосс или тейк-профит

Уведомления приходят для **всех** типов сигналов (BUY, SELL и HOLD), чтобы вы всегда были в курсе текущего состояния рынка.

### Формат сообщения:
```
Trading Signal [BUY]

Symbol: BTCUSDT
Strategy: RSI
Price: 75,301.28
Timeframe: 1h
Confidence: 75.0%
Time: 2026-04-14T17:59:22

[Auto] RSI | BUY | $75301.28 | 1h
```

## 9. Автоматический мониторинг

Система может автоматически отслеживать рынок и генерировать сигналы без вашего участия.

### Как включить (3 шага):

**Шаг 1.** Создайте стратегию:
- Перейдите на страницу **Стратегия**
- Нажмите **"+ Новый пресет"** внизу страницы
- Выберите монету, стратегию, таймфрейм, стоп-лосс и тейк-профит
- Нажмите **Создать**

**Шаг 2.** Активируйте стратегию:
- На странице **Стратегия** — включите переключатель **АВТО-ТОРГОВЛЯ** рядом с пресетом
- Или на странице **Настройки → Мониторинг стратегий** — включите зелёный toggle

**Шаг 3.** Настройте интервал:
- На странице **Настройки → Monitoring Interval**
- Установите интервал (например, 10 секунд или 1 минута)
- Нажмите **Apply** или **Start**

### Что происходит автоматически:
С заданным интервалом (по умолчанию 60 секунд, настраивается от 5 секунд до 1 часа) система:
1. Проверяет все включённые стратегии
2. Загружает свежие данные с Binance
3. Рассчитывает индикаторы
4. Генерирует сигнал (BUY / SELL / HOLD)
5. Сохраняет сигнал в базу данных
6. **Отправляет уведомление в Telegram** (для всех типов сигналов)
7. Если сигнал BUY или SELL — открывает виртуальную сделку
8. Проверяет открытые сделки на стоп-лосс и тейк-профит:
   - Если цена упала ниже стоп-лосса — закрывает сделку с убытком
   - Если цена выросла выше тейк-профита — закрывает сделку с прибылью
   - Отправляет уведомление о закрытии в Telegram

### Пример сценария:
```
1. Вы создали стратегию: RSI + BTCUSDT + 1h + SL 2% + TP 3%
2. Включили toggle → стратегия активна
3. Установили интервал 10 секунд → нажали Apply
4. Каждые 10 секунд в Telegram приходит:
   - "Trading Signal [HOLD] - BTCUSDT - RSI - $75,301" 
   - или "Trading Signal [BUY] - BTCUSDT - RSI - $74,100"
5. При BUY — автоматически открывается виртуальная сделка
6. Когда цена вырастет на 3% — сделка закроется с прибылью
```

### Виртуальные сделки
Все сделки — **виртуальные** (бумажная торговля). Реальные деньги не используются. Это позволяет:
- Оценить эффективность стратегии в реальных рыночных условиях
- Отслеживать историю и P/L
- Сравнивать стратегии между собой

## 12. Часто задаваемые вопросы

### Откуда берутся данные о ценах?
Данные получаются в реальном времени с биржи **Binance** через их публичный API. Цены обновляются каждые 10 секунд.

### Можно ли потерять деньги?
Нет. Платформа НЕ совершает реальные сделки. Все сделки виртуальные (бумажная торговля). Платформа предназначена для аналитики и обучения.

### Какую стратегию выбрать?
- **Для начинающих** — RSI или Combined (комбинированная)
- **Для тренда** — MACD или EMA
- **Для диапазонной торговли** — Bollinger Bands
- **Для максимальной надёжности** — Combined (голосование всех стратегий)

### Что такое "уверенность" (confidence)?
Это числовая оценка (0-100%) насколько сильно индикаторы подтверждают сигнал. Чем выше — тем увереннее сигнал.

### Почему мне не приходят уведомления в Telegram?
Проверьте по порядку:
1. Вы написали `/start` боту **@aiturgan_cti_bot** в Telegram
2. Правильно ввели Chat ID в **Настройки → Telegram** и нажали **Сохранить**
3. Нажмите **Тест** — если тестовое сообщение пришло, подключение работает
4. Есть хотя бы одна **активная стратегия** (зелёный toggle в Настройки → Мониторинг стратегий)
5. Мониторинг **запущен** (Настройки → Monitoring Interval → статус "Active")
6. Если всё выше ОК но сигналы не приходят — создайте новую стратегию на странице Strategy → "+ New Preset" и включите её

> **Самая частая причина:** стратегии не созданы или не активированы. Мониторинг работает, но без активных стратегий ему нечего проверять.

### Что означает HOLD?
**HOLD (Ожидание)** — индикаторы показывают неопределённую ситуацию. Рекомендуется не открывать позицию и дождаться более чёткого сигнала.

### Как часто обновляются данные?
- Цена в верхней панели — каждые 10 секунд
- Автоматический мониторинг — настраивается (от 5 секунд до 1 часа, по умолчанию 60 секунд)
- Графики — при загрузке страницы или смене монеты/таймфрейма

### Можно ли изменить частоту уведомлений?
Да. Перейдите в **Настройки → Monitoring Interval**. Установите нужный интервал в секундах (минимум 5s) и нажмите **Apply**. Есть быстрые кнопки: 10s, 30s, 1m, 5m.

### Приходят ли уведомления при HOLD?
Да. Система отправляет уведомления для **всех** типов сигналов — BUY, SELL и HOLD. Это позволяет отслеживать текущее состояние рынка в реальном времени.

### Можно ли экспортировать данные?
Да. На страницах **Бэктест** и **Отчёты** есть кнопки **ЭКСПОРТ CSV** для сохранения данных в формат CSV (открывается в Excel, Google Sheets).
